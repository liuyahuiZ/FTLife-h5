###后端api koa2
> 后端接口是koa2开发的一个小后端，有图片管理等功能

###注意事项


## Build Setup

``` bash
# clone code
git clone git@github.com:liuyahuiZ/FTLift-h5.git
cd wetalk

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
